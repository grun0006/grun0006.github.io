FAQ

Hoe kan je een document printen vanaf je laptop bij de HZ?

Ga je HZ website. Log daar in met je schoolaccount. Je kan een bestand uploaden bij WebPrint of de printer installeren bij MobilityPrint.


Hoe kan je een document scannen en naar je laptop sturen bij de HZ?

Ga naar een scanner/printer. Log in. Druk op Scan. Leg je papieren waar ze horen en druk op Start. Log daarna weer uit.

Hoe kan je een project ruimte boeken?

Op de myhz website staat op de homepagina een knopje genaamd Selfservice Portal waar je een project ruimte kan boeken als je erop klikt.


Wat zijn de instructies als je je auto op de HZ parkeerplaats wil parkeren?

Met de studentenpas mag je parkeren op het parkeerterrein voor de HZ.
